# arq-2-unq
Repositorio para trabajos practicos y entregas de la materia Arquitectura II de la Universidad Nacional de Quilmes

## Estructura del Repositorio

Actualmente, el repositorio cuenta con los siguientes proyectos:

### 1. [WeatherFlow](./weather_flow/)
*Trabajo Práctico 1*
Aplicación de servicios meteorológicos construida como API REST con Elixir, Phoenix y MongoDB, estructurada bajo los principios de Arquitectura Hexagonal y Domain-Driven Design (DDD). 

Dirígete al [README interno del proyecto](./weather_flow/README.md) para obtener más detalles sobre su arquitectura, dependencias e instrucciones para correrlo de forma local.
